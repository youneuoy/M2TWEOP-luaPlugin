LuaPlugin ImGui bindings

Welcome to the LuaPlugin ImGui bindings. This bindings is being developed to support M2TWEOP-luaPlugin

Enjoy!

Notes

  • ImGui lua binding is based on youneuoy`s fork of this repository.
  • This binding is adapted for the needs of the M2TW Engine Overhaul Project

Documentation

You can find all the supported functions and overloads below.

Windows

  -- ImGui.Begin(...)
  -- Parameters: text (name), bool (open) [O], ImGuiWindowFlags (flags) [O]
  -- Returns A: bool (shouldDraw)
  -- Returns B & C: bool (open), bool (shouldDraw)
  -- Overloads
  shouldDraw = ImGui.Begin("Name")
  open, shouldDraw = ImGui.Begin("Name", open)
  open, shouldDraw = ImGui.Begin("Name", open, ImGuiWindowFlags.NoMove)

  -- ImGui.End()
  ImGui.End()
